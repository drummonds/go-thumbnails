Visual examples of go-thumbnails output. These images are generated by `cmd/gendocimages` and match the test suite.

## Rendering Styles

The composite style shows up to 4 pages side-by-side. Documents with more than 4 pages get a "+" indicator. Single-page documents render as a single resized page.

## Per-page Rendering

The page-level API (`RenderPages`, `RenderPage`, `ResizePage`) extracts individual pages from multi-page documents.

```go
// All pages
pages, err := thumbnails.RenderPages("doc.pdf")
for _, p := range pages {
    resized := thumbnails.ResizePage(p.Image, 128)
    // p.PageNum (1-based), p.PageCount
}

// Single page
page, err := thumbnails.RenderPage("doc.pdf", 3)
resized := thumbnails.ResizePage(page.Image, 128)
```
